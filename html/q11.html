題目在問什麼

這題不是要你背名詞，而是要你看出題幹正在考：

Random Search 相較 Grid Search 在高維超參數空間中較有效率，避免網格搜尋的維度災難。

題幹線索可以先抓這一句：

相較於GridSearch，RandomSearch在超參數調整上具備哪一項主要優勢？

所以解題順序是：

  1. 先判斷題目描述的是哪一種任務或問題。
  2. 再找最能直接處理這個問題的模型、指標、方法或程式碼位置。
  3. 最後排除只是「看起來相關」、但沒有真正回答題幹要求的選項。

為什麼答案是 D

Grid Search 容易遭受維度災難（Curse of Dimensionality），當參數變多時計算量呈 指數成長。Random Search 在高維參數空間中，能以較少的嘗試次數找到接近最佳解的參數組 合，效率較高。
