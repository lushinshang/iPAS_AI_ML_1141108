第 45 題解析：VGG16 遷移學習與凍結參數

思維路由：林守德的模型結構辨識 + 蔡宗翰的程式碼劇本化。

答案：B

題目在問什麼

這題不是考你會不會寫 Python，而是在考你能不能看懂 VGG16 的「身體部位」。

VGG16 大致分成兩段：

VGG16 部位 在 PyTorch 裡的名稱 功能
卷積層區塊 model.features 抽影像特徵，例如邊緣、紋理、形狀
全連接分類器 model.classifier 根據特徵做最後分類

題目說：

凍結卷積層的參數，只訓練最後全連接層 classifier

所以正確動作應該是：

  1. 找到卷積層：model.features
  2. 把卷積層凍結：requires_grad = False
  3. 換掉最後分類輸出層：model.classifier[6] = Linear(4096, 10)

這正是 B。

for param in model.features.parameters():
    param.requires_grad = False
model.classifier[6] = torch.nn.Linear(4096, 10)

選項快速判斷

A 錯：凍結整個模型

for param in model.parameters():
    param.requires_grad = False

model.parameters() 代表整個 VGG16，包括卷積層和分類器都被凍結。雖然後面換了 classifier[6]，但它只會訓練新換上的最後一層，不符合「只凍結卷積層，訓練 classifier」的完整語意。

B 對：只凍結卷積層

for param in model.features.parameters():
    param.requires_grad = False

看到 features 就要想到：卷積特徵抽取區。這正是題目要求凍結的部分。

C 錯：凍結 classifier

for param in model.classifier.parameters():
    param.requires_grad = False

題目要訓練 classifier，這個選項卻把 classifier 凍結，方向完全相反。

D 錯：寫法不正確

model.requires_grad = False

這只是幫 model 物件加一個屬性，不會真正逐一凍結裡面的權重參數。凍結要針對 parameters() 裡的每個參數設定。

考場秒殺法

看到題目關鍵字：

凍結卷積層

就找：

model.features.parameters()

看到：

訓練 classifier

就不要選會凍結 model.classifier.parameters() 的選項。

快速口訣：

凍結卷積找 features，訓練分類別凍 classifier。
